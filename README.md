
# Kytos-ng.github.io

This repository hosts certain parts of Kytos-ng's documentation with `jekyll`

## Install dependencies locally in project folder

To have the dependencies installed in the same folder as this project, run the following command

```
bundle config set --local path vendor/bundle
```

## How to try it out locally

- Make sure you have [`bundler`](https://bundler.io/) and `ruby` installed

```
bundle install
bundle exec jekyll serve
```

- Once `jekyll` is running and serving, you should be able to load the docs on http://127.0.0.1:4000/
